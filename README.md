# PIANO ELECTRONICO
#PAVONI MATIAS - FRIGERIO JUAN CRUZ

Nuestro proyecto se basa en un piano electrónico, este consiste en 7 pulsadores que simulan las teclas del piano, donde las frecuencias de cada pulsador forman parte de las frecuencias que componen la octava central del piano sin incluir los semitonos. Además, nuestro proyecto cuenta con distintas funcionalidades que nos permiten cambiar de octava y también grabar una secuencia de notas reproducidas, esta grabación puede tener el tamaño que deseemos considerando las limitaciones físicas(cantidad de almacenamiento disponible).
Para nuestro trabajo utilizamos las herramientas Arduino IDE, Processing y Tinkercad.
